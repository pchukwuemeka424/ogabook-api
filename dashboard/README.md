# Dashboard Folder

This folder is for organizing dashboard-related files, components, and assets.

## Structure

- Dashboard views are located in `/views/dashboard.ejs`
- Dashboard route is `/dashboard`
- After login, users are redirected to `/dashboard`

## Dashboard Features

- Overview statistics (Users, Subscriptions, Sales, Tables)
- Quick action buttons
- Real-time data loading
- Responsive design

## Access

- URL: `http://localhost:3000/dashboard`
- Requires authentication
- Root path (`/`) redirects to `/dashboard`
